DỀ TÀI 25: XÂY DỰNG PHẦN MỀM QUẢN LÝ KHÁCH SẠN
CHƯƠNG 1: TỔNG QUAN DỰ ÁN 

1.1.	Giới thiệu đề tài 
Phần mềm Quản lý Khách sạn là một ứng dụng desktop được xây dựng bằng ngôn ngữ lập trình Python, sử dụng thư viện PyQt6 để xây dựng giao diện đồ họa (GUI) và SQLite làm cơ sở dữ liệu. Phần mềm giúp quản lý toàn bộ hoạt động của khách sạn từ quản lý phòng, đặt phòng, check-in/check-out đến thanh toán và thống kê doanh thu.
1.2.	Mục tiêu dự án 
•	Xây dựng hệ thống quản lý khách sạn hoàn chỉnh, đáp ứng nhu cầu thực tế
•	Ứng dụng kiến thức PyQt6 (GUI), SQLite (Database) vào sản phẩm thực tiễn
•	Quản lý mã nguồn trên GitHub

1.3.	Phạm vi dự án 
Phần mềm phục vụ nhân viên lễ tân và quản lý khách sạn với các chức năng:
•	Quản lý danh mục phòng và tình trạng phòng
•	Quản lý thông tin khách hàng
•	Xử lý đặt phòng, check-in, check-out
•	Quản lý dịch vụ bổ sung
•	Thanh toán và xuất hóa đơn
•	Thống kê báo cáo doanh thu

1.4.	Công nghệ sử dụng 
Python-3.14-Ngôn ngữ chính
PyQt6-6.x-Xây dựng giao diện đồ họa (GUI)
SQLite-Built-in-Lưu trữ dữ liệu cục bộ
GitHub--Quản lý mã nguồn và cộng tác nhóm
